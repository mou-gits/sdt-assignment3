# Programmable Remote Control System

A Java Swing application that simulates a programmable remote control for a smart home.  
Users can assign commands to remote buttons, control devices, and visualize device states on a floor plan.

---

## Overview

This system provides:

- A **floor plan panel** showing device states (AC, Garage Door, Indoor Light, Outdoor Light)
- A **14‑button programmable remote**
- A **command assignment panel** with dropdown menus
- A **JSON‑based persistence layer** for saving button mappings
- A backend using the **Command Pattern**
